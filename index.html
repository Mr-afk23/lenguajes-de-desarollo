Html

¿ Que es html?

HTML es un lenguaje de marcado que define la estructura de una página web mediante etiquetas. No es un lenguaje de programación, sino un lenguaje que organiza y jerarquiza el contenido de un sitio.

¿ Para que sirve ?

Define la estructura de las páginas web. Permite incluir texto, imágenes, enlaces, tablas, formularios y otros elementos. Facilita la conexión con hojas de estilo (CSS) y scripts (JavaScript) para mejorar la experiencia del usuario.

Java script

¿ Que es Java script

JavaScript es un lenguaje de programación que permite agregar interactividad a las páginas web. Es uno de los lenguajes más utilizados en el desarrollo web.

¿ Para que sirve ?

Agregar efectos visuales y animaciones. Validar formularios antes de enviarlos al servidor. Crear aplicaciones web dinámicas. Manejar eventos (clics, desplazamientos, cambios en formularios). Conectar con servidores sin recargar la página (AJAX y Fetch API).

CSS

¿ Que es css?

CSS es un lenguaje de estilos que define la apariencia de los elementos HTML. Permite diseñar las páginas web, haciendo que se vean atractivas y bien organizadas.

¿ Para que sirve ?

Define colores, fuentes y tamaños de texto. Controla el diseño de la página con márgenes, alineaciones y posicionamientos. Facilita la creación de sitios web adaptables (responsive design). Separa la estructura (HTML) del diseño (CSS), permitiendo cambios rápidos y globales en el estilo del sitio.

FrameWork

¿Que es FrameWork

Adobe Fireworks es un software de diseño gráfico enfocado en la creación de gráficos para la web y aplicaciones digitales. Fue desarrollado por Macromedia y luego adquirido por Adobe. Aunque Adobe dejó de actualizarlo en 2013, sigue siendo utilizado por algunos diseñadores.

¿Para que sirve

Diseñar interfaces de usuario para páginas web y aplicaciones. Crear prototipos interactivos de sitios web. Editar imágenes con herramientas avanzadas de vector y píxeles. Exportar gráficos en formatos optimizados para la web, como PNG, JPG y GIF.
